Bonus Fitur
Program ini menambahkan fitur *Processed Queue* menggunakan `Queue<Product<?>>` yang memungkinkan pengguna 
menyimpan dan menampilkan urutan produk yang telah diproses seperti pada sistem pengemasan atau pengiriman barang.

Fitur ini memanfaatkan:
- **Queue (LinkedList)** sebagai FIFO antrian.
- **Generic Product<?>** agar tetap fleksibel untuk segala tipe kategori.
